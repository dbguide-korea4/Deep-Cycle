## How to use

1. setup.py를 실행하여 환경을 맞춰준다.
1. DpCycle/logs폴더에 weight(h5 파일)를 넣는다. [model weight download](https://drive.google.com/open?id=1xjmrq8s6Vrm_rfqV0S89cX6CjA8RZUC5)
1. DpCycle 포로젝트를 vsCode 혹은 pyCharm을 이용해서 flask로 실행
